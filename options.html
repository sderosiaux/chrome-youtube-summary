YouTube Summary Settings

Configure your OpenAI API key and customize the summary prompt

OpenAI Configuration

Your API key is stored locally and never shared. Get one at OpenAI Platform

Summary Prompt

Leave empty to use the default prompt. The prompt will be combined with the video title, channel, and transcript.

Usage Instructions

  1. Set your OpenAI API key above
  2. Optionally customize the summary prompt
  3. Navigate to any YouTube video
  4. Click the extension icon or use the keyboard shortcut
  5. Wait for the AI to generate your summary

Note: The extension works best with videos that have captions/transcripts available.
